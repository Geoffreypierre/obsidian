# Site Codéin

Ce ticket concerne l'apport des accès aux contenus connexes sur les pages du site. Se manifestant sous la forme d'une file de Tag concaténée à la file d’Ariane. Cette file serait tronquée selon la largeur de l'écran, l'affichage commencerait à gauche à partir du premier éléments et les éléments tronqués serait accessible successivement via une flèche de divulgation.
